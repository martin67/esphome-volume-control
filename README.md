# ESPHome Project Template

Template for ESPHome-based hardware projects with:

- ESPHome firmware
- Schemdraw wiring diagrams
- GitHub Actions SVG generation
- Structured hardware + enclosure files

## Structure

- `firmware/` → ESPHome YAML
- `docs/` → Schematics + documentation
- `hardware/` → KiCad / electronics design
- `enclosure/` → 3D models
